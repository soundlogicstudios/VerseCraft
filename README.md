# VerseCraft Starter v0.0.01

This is a clean, panel-first starter for VerseCraft:
- No modals required
- Full-bleed pre-rendered art panels
- Navigation wired through invisible hitboxes (CSS %)
- iOS Safari friendly defaults

## First Run Checklist
1. Add your WebP assets into:
   - backgrounds/ui/menu/menu_nosave.webp
   - backgrounds/ui/settings/settings.webp
   - backgrounds/ui/skins/library/backgrounds/library.webp
2. Commit + push to GitHub Pages
3. Open: `https://<your-username>.github.io/<repo>/#menu`
4. Toggle hitbox outlines: click **Debug** or add `?debug=1`

## GitHub Pages
Set Pages to:
- Branch: `main`
- Folder: `/ (root)`

## Alignment
Edit `styles/ui/hitboxes.css` and adjust top/left/width/height (%).
